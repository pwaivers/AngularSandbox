# AngularSandbox
AngularJS Sandbox and Seed Application

## Setup
You should already have `node` and `bower` installed.

First, install the node packages
```
> npm install
```

Then, install the bower packages
```
> bower install
```

Lastly, run `grunt`
```
> grunt
```

## What is included in this project
* AngularJS
* Bootstrap
* Grunt
